==========
 Projects
==========

Projects provide generel information about a software project (a.k.a.
product).

>>> import metatracker.project
>>> project = metatracker.project.Project('metatracker buglist',
...     'The list of bugs for metatracker should be visible on the dashboard.')
>>> project
<metatracker.project.Project object at 0x...>

We check if the bug really imports it's interface:

>>> import zope.interface.verify
>>> import metatracker.interfaces
>>> zope.interface.verify.verifyObject(
...     metatracker.interfaces.IProject, project)
True
