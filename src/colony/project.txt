==========
 Projects
==========

Projects provide generel information about a software project (a.k.a.
product).

>>> import colony.project
>>> project = colony.project.Project('colony buglist',
...     'The list of bugs for colony should be visible on the dashboard.')
>>> project
<colony.project.Project object at 0x...>

We check if the bug really imports it's interface:

>>> import zope.interface.verify
>>> import colony.interfaces
>>> zope.interface.verify.verifyObject(
...     colony.interfaces.IProject, project)
True
